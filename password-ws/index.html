パスワードチャレンジ 🔐

ストップウォッチで測りながら、7つの条件をすべて満たすパスワードを作ろう！
スタートを押してから入力開始！

00:00.00

条件リスト

  • ✗
    8文字以上 例: password123
  • ✗
    大文字（A〜Z）を含む 例: Password → ✓
  • ✗
    数字（0〜9）を含む 例: Pass1word → ✓
  • ✗
    記号（! @ # $ % など）を含む 例: Pass!word → ✓
  • ✗
    ¥（円記号）を含む Apple Accountの実際の要件！
  • ✗
    全角「Ａ」と半角「l」が連続して登場する 例: Ａl ← この2文字の並びを含む（全角Aのあとに半角l）
  • ✗
    🍣（寿司の絵文字）を含む そのままコピペしてもOK！
🎰

全条件クリア！！
